## Summary
A Tech News website which displays articles for a user to click on and get more information about the story. 

## Tech 
React.Js
NewsApi

## Setup 
1. Clone repo to local enviroment
2. cd into TakeHome
3. run npm install
4. run 'npm run dev'
5. Enjoy!
